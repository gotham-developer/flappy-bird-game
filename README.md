# Flappy Bird Clone

A clean, smooth, and optimized **Flappy Bird-style game** built using **HTML5**, **CSS3**, and **vanilla JavaScript**.  
The game provides a polished user experience with fluid bird controls, intuitive restart mechanics, and a responsive pixel-art environment.

## Features

- Jump using **Arrow Up**, **Space**, **Enter**, or **Mouse Click**
- Restart using **Space**, **Enter**, **Click**, or **Touch**
- Smooth gravity-based bird movement
- Dynamic pipe generation with balanced spacing
- Accurate scoring system with sound effects
- Mobile-friendly controls
- Lightweight structure without external libraries

## Technology Stack

- **HTML5** for layout and rendering
- **CSS3** for styling and visual polish
- **JavaScript (Vanilla)** for physics, input handling, and game loop logic
- No frameworks or build tools required

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/flappy-bird-clone.git
cd flappy-bird-clone
```

### 2. Run the game locally

No installations required.  
Simply open:

```
index.html
```

in any modern browser.

### 3. Optional: Run a lightweight dev server

```bash
npx serve .
```

or using Python:

```bash
python3 -m http.server
```

Access the game at:

```
http://localhost:8000/
```
